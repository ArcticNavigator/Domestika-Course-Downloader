Domestika Downloader

How to use

  1. Install Cookie-Editor : Add the Cookie-Editor extension to Chrome from the Chrome Web Store
  2. Log in to Domestika : Open domestika.org in Chrome and sign in to your account
  3. Open Cookie-Editor : Click the Cookie-Editor extension icon in your Chrome toolbar
  4. Copy your Session Cookie : Find _domestika_session in the list, click it to expand, copy the Value field, and paste it into the Session Cookie field below
  5. Copy your Credentials : Find _credentials_ in the list, copy its value and paste it into the Credentials field below
  6. Add courses : Paste a Domestika course URL and click + Add (or press Enter). Repeat for multiple courses
  7. Start : Click ▶ Start Downloads and watch the progress below
  8. Controls : Use ⏸ Pause, ⏭ Skip, or ⏹ Stop at any time while downloading

💡 Credentials are only needed to download the Final Project video. You can leave it blank if you don't need it.

Settings reference

  • Subtitle Language : Language code for subtitles (e.g., en, es). Make sure the language is available for the course before setting.
  • Transcode to H.265 (HEVC) : When enabled, automatically transcodes the downloaded file to H.265/HEVC if it isn't already. Reduces file size while maintaining quality. Disable to keep the original format.
  • Use GPU encoder (NVENC) : Only relevant when Transcode to H.265 is enabled.
    • Checked : Uses NVIDIA GPU hardware acceleration. Faster, but requires an NVIDIA GPU with NVENC support.
    • Unchecked : Uses CPU-based encoding (libx265). Works on any machine, but slower.
  • Max Parallel Downloads : Number of downloads and transcodes to run at the same time. Recommended: 2-4. Higher values speed things up but use more CPU and memory.
